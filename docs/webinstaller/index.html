Copy this FAP file to your Flipper (i.e. to apps/GPIO using qFlipper). You also might need this FAP for the Stream mode. Next, reset ESP32CAM with pin IO0 connected to GND and upload the firmware pressing Install.

Note: If you are using your Flipper to upload the firmware, remember to go to GPIO/USB-UART Bridge.

Enjoy!
